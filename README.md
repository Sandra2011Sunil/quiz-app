# Quiz App

An interactive Computer Science quiz app built with 
HTML, CSS and JavaScript.

## Features
- 10 CS multiple choice questions
- Real time score tracking
- Progress bar
- Color coded correct and wrong answers
- Final score with performance message
- Restart functionality
- Responsive glassmorphism UI

## Tech Stack
- HTML5, CSS3, JavaScript (ES6+)

## How to Run
1. Clone the repo
2. Open index.html in browser
3. Click Start Quiz and test your knowledge!
